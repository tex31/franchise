Page de login

Tout utilisateur souhaitant accéder aux fonctionnalités de l'application doit s'authentifier sur cette page.

  • pour valider le choix de fonction et accéder au tableau de bord du fonctionnalité choisie

Comptabilité matières

La comptabilité matières présente 7 menus :

  • Tableau de bord (Administration)
  • Saisie des référentiels
  • Saisie des matériels
  • Détenteur
  • Décharge
  • Suivi et éditions des états règlementaires
  • Validations

N.B : La disponibilité de ces menus dépend de la fonction de l'utilisateur courant. (Voir Accès utilisateur)

4.1 - Administration et Opérations

La page par défaut de l'application dans cette comptabilité est le tableau de bord ou administration, ce menu est disponible pour tout utilisateur quelle que soit sa fonction.

Pour chaque utilisateur, ce tableau présente la liste des opérations effectuées par l'utilisateur courant. Les détails sur les opérations effectuées figurant dans le tableau sont :

  • Date : date de l'opération
  • Poste : poste de l'opérateur
  • L'état : État actuel de l'opération, il existe 4 types d'état
    • ACCEPTEE
    • EN ATTENTE
    • REFUSEE
    • A MODIFIER
  • Opérateur : Immatriculation de l'opérateur
  • Types : Il existe 7 types d'opérations disponibles :
    • Entrée de matériels
    • Sortie de matériels
    • Saisie des référentiels
    • Attribution
    • Détachement
    • Entrée d'un article
    • Sortie d'un article
  • Pièces justificatives : lien pour générer un fichier concernant la justification de l'opération effectuée.

Les types d'opérations : Entrée de matériels, Attribution et détachement possèdent des pièces justificatives. Voir la section Édition des états règlementaires pour remplir le formulaire

Les filtres disponibles sont un filtre par état, un filtre par type d'opération et un filtre pour une recherche globale

4.2 - Saisie des référentiels

Fonctionnalités destinées uniquement aux administrateurs Service Informatique (SI) et Service Equipement (SE) pour l'enregistrement des données relatives aux référentiels. Toutes les rubriques disponibles sont affichées au niveau de l'interface générale .

Chaque tableau de référentiel contient Qui permettra à l'administrateur de procéder à l'ajout de nouveau type de référentiel.
Chaque tableau contient également les élements d'un type de référentiel (ex: liste de nomenclature ou liste de type de matériels).
Dans chaque tableau, l'utilisateur courant peut modifier ou supprimer des référentiels.

Un bouton filtre est disponible en bas de page pour faciliter l'accés direct au tableau de réferentiel

N.B : Aucun doublon n'est autorisé dans les saisies des référentiels. Le message d'erreur ci-dessous apparait en cas de doublon detecté..

Ci-dessous les différents types de réferentiels utilisés dans le cadre de l'application . L'administrateur peut modifier , rajouter ou supprimer des élements au besoin.

  • 4.2.1 - Nomenclature

      • Cette interface est utilisée pour ajouter une nouvelle nomenclature, 2 champs sont à remplir :

    • Désignation : texte définissant la nomenclature ex: Combustibles et Luminaires
    • Nomenclature : code unique selon la nomenclature officielle ex:1

    Tous ces champs sont obligatoires, un message d'erreur apparaît dans le cas contraire

  • 4.2.2 - Types de matériels

      • Cette interface est utilisée pour ajouter un nouveau type de materiel, 3 champs sont à remplir :

    • Désignation : texte définissant le type de matériel ex:Lampe tube fluorescent
    • Code type matériel : Code unique composé de : quatre premières lettres de la désignation du matériel sous forme capital. ex: ARMB
    • Catégorie : Catégorie de la nomenclature associé au type de matériel à choisir

    Tous ces champs sont obligatoires, un message d'erreur apparaît dans le cas contraire

  • 4.2.3 - Etat de matériels

      • Cette interface est utilisée pour ajouter une qualification de l'état de matériel

    • Désignation : texte définissant l'état du matériel

    Ce champ est obigatoire, une message d'erreur apparaît dans le cas contraire

  • 4.2.4 - Agents (Utilisateurs)

    • Cette interface est utilisée pour créer un nouvel utilisateur, 7 champs sont à remplir :

    • Nom et prénom : Champ de type lettre, contenant le nom et prénom du nouvel agent
    • Immatriculation : Champ de type numérique uniquement, contenant le matricule du nouvel agent tel que la longueur minimale du matricule est 6
    • Fonction : Menu déroulant les différentes fonctions existantes ex: SISE -GAC - DC - Agent
    • Mot de passe : Champ de type Alphanumerique d'une longueur minimale du mot de passe de 8 contenant le mot de passe de l'agent.
    • Retaper mot de passe : Champ de type Alphanumerique, contenant le mot de passe identique au champ précédent pour la confirmation
    • Direction : Menu déroulant les directions existantes au sein de la douane pour choisir la direction à laquelle l'utilisateur appartient
    • Poste : Menu déroulant les différents postes existants

    Tous ces champs sont obligatoires, un message d'erreur apparaît dans le cas contraire

  • 4.2.5 - Marques

      • Cette interface est utilisée pour ajouter une nouvelle marque

    • Désignation : texte définissant la marque

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.6 - Motifs d'aquisitions

      • Cette interface est utilisée pour ajouter un nouveau motif d'acquisitions de matériels ou équipements

    • Désignation : texte définissant le motif d'acquisition

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.7 - Motifs de décharge

      • Cette interface est utilisée pour ajouter un nouveau motif de décharge

    • Désignation : texte définissant le motif de décharge

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.8 - Financement

      • Cette interface est utilisée pour ajouter une nouvelle source de financement

    • Désignation : texte définissant le financement

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.9 - Poste

      • Cette interface est utilisée pour ajouter un nouveau poste

    • Désignation : texte définissant le nouveau poste

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.10 - Direction

      • Cette interface est utilisée pour ajouter une nouvelle direction

    • Désignation : texte définissant la nouvelle direction
    • Code direction : code unique représentant la direction
    • Intitulé : Intitulé de la direction qui peut changer au fur et au mesure d'une nouvelle assignation
    • Localité : Localisation de la direction
    • Budget : Budget correspondant à la direction - utilisée dans les états règlementaires (voir cette section)
    • Subdivision du chapitre :Subdivision du chapitre correspondant à la direction - utilisée dans les états règlementaires (voir cette section)

    Tous ces champs sont obligatoires, un message d'erreur apparaît dans le cas contraire

  • 4.2.11 - Motif de sortie

      • Cette interface est utilisée pour ajouter un nouveau motif de sortie

    • Désignation : texte définissant le motif de sortie (ex: Affection/Perte/Départ)

    Ce champ est obligatoire, un message d'erreur apparaît dans le cas contraire

  • 4.2.12 - Devise

      • Cette interface est utilisée pour ajouter une nouvelle devise de la république

    • Désignation : texte définissant la nouvelle devise de la république
    • Date du changement : Date définissant le debut de l'utilisation de la nouvelle devise de la république

    Tous ces champs sont obligatoires, un message d'erreur apparaît dans le cas contraire

4.3 - Saisie des matériels existants

Ce menu est uniquement destiné au Dépositaire Comptable (DC) pour pouvoir ajouter et visualiser les matériels existants dans la direction.

Seuls les utilisateurs Administrateur,Administrateur général ont accés à ce menu en cas de modification des données de ces matériels

Le tableau contenant la liste des matériels présente les informations suivantes:

  • ID : Identifiant du matériel
  • Référence : Référence associée au matériel
  • Numéro de Série : Numéro de série du matériel
  • Nomenclature : Nomenclature associée au matériel
  • Etat : Etat du matériel
  • Marque Marque du matériel
  • Dépositaire Nom du dépositaire
  • Code Code associé au matériel composé de: code de la Direction/code du type de matériel / Numéro selon le type de matériel / Année d'acquisition ex: DGD/2/ORD1/4/2011
  • Détails Champ contenant 2 icone, l'un de visualisation et l'autre de modification

Des filtres sur l'ID, la référence, le numéro de série et la nomenclature sont disponibles pour faciliter les recherches de matériels

Image : Associer une photo au matériel à ajouter

  • Pour les upload, les types de fichiers acceptés sont : gif, jpeg, png, txt, pdf, doc, docx

  • Le bouton permet d'ajouter un fichier à uploader

Nomenclature : Rempli automatiquement selon le type de matériel choisi

Marque : Liste déroulante des marques disponibles et possibilité d'ajouter une nouvelle

Prix unitaire : Nombre décimal au format monétaire

Caractéristiques : Texte descriptif du nouveau matériel

Autre : Autre information à declarer sur le nouveau matériel

Année acquisition : Champ à valeur entier au format date(année) décrivant l'année d'acquisition du matériel

Origine : Origine (ou provenance) du matériel

Etat : Liste déroulante des différents états de matériels

Numéro de série : Libellé du numéro de série du matériel. Dans le cas où le matériel n'a pas de numéro de série alors la valeur à assigner doit être équivalent à la référence

N.B : Pour ajouter le(s) matériel(s), il est obligatoire pour l'utilisateur de cliquer sur le bouton "Ajouter nombre par type" avant de l'enregistrer. Tous les matériels à ajouter doivent être contenus dans tableau à la fin du formulaire avec sa référence et son numéro de série.

4.4 - Prise en charge

Ce menu est uniquement destiné au Dépositaire Comptable (DC) pour pouvoir visualiser ou modifier les matériels prise en charge dans une facture. L'utilisateur peut également ajouter les nouveaux matériaux à prendre en charges à partir de la facture reçue. Cette prise en charge devra être validée par le GAC

Seuls les utilisateurs Adminsitrateur,Administrateur général ont accés à ce menu en cas de modification des données qui ont été déjà validés par le GAC de la direction

L'interface de ce menu contient 2 tableaux dont la liste des matériels prise en charge acceptée et la liste des matériels prise en charge non acceptée

Concernant les détails, les modifications, et l'ajout, les interfaces sont similaires à ceux de la saisie des matériels existants mais des champs supplémentaires sont à remplir et ces matériels nécessitent une validation.

4.5 - Attribution

Ce menu est uniquement destiné au Dépositaire Comptable (DC) pour pouvoir attribuer un matériel à un agent (détenteur) de sa direction.

Pour attribuer un matériel à un agent, les champs(obligatoires) suivants sont à remplir :

    Matériel : liste déroulante des matériels disponibles (à partir de son numéro de série)

    Nomenclature : Rempli automatiquement selon le matériel choisi

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Détenteur : Liste déroulante des agents pour être le nouveau détenteur du matériel choisi.

    Nom : Rempli automatiquement selon le détenteur choisi

  • La soumission de l'opération d'attribution aux GAC de la Direction se fait en cliquant sur le boutton Attribuer

    • N.B : Les champs Matériels et Détenteur sont obligatoires, un message d'erreur apparaît dans le cas contraire

4.6 - Détachement

Ce menu est uniquement destiné au Dépositaire Comptable (DC) pour pouvoir détacher un matériel détenu par un agent de sa direction.

Pour détacher un matériel détenu par un agent, les champs(obligatoires) suivants sont à remplir :

    Détenteur : Liste déroulante des agents (représenté par sa matriucle) détenteur de matériel.

    Nom : Rempli automatiquement selon le détenteur choisi

    Matériel : Liste déroulante des matériels disponibles pour pouvoir sélectionner le matériel retenu

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Codification : Rempli automatiquement selon le matériel choisi

    Motif : Liste déroulante des différentes motifs de détachement

  • La soumission du détachement aux validateurs (GAC) se fait en cliquant sur le bouton Détacher

    • N.B : Les champs Détenteur, Matériel et Motif sont obligatoires, un message d'erreur apparaît dans le cas contraire

4.7 - Sortie de matériel (décharge)

Ce menu est uniquement destiné au Dépositaire Comptable (DC) pour pouvoir décharger ou sortir un matériel détenu selon un motif.

Pour procéder à une sortie de matériel vers une autre Localité/Direction, les champs(obligatoires) suivants sont à remplir :

    Motif de sortie : Liste déroulante des choix possibles de motif de sortie d'un matériel

    Matériel : Liste déroulante des matériels

    Marque : Rempli automatiquement selon le matériel choisi

    Référence : Rempli automatiquement selon le matériel choisi

    Numéro de série : Rempli automatiquement selon le matériel choisi

  • La soumission de la décharge aux validateurs (GAC) se fait en cliquant le bouton Décharger

    • N.B : Le champ Matériel est obligatoire, une message d'erreur apparaît dans le cas contraire

4.8 - Traçabilité

Ce menu est accéssible pour les personnels du Gestionnaires d'Activités (GAC), Administrateur,Administrateur général, Consultation, Consultation générale . Ce menu présente un tableau de bord de toutes les opérations de la direction.

Le tableau ci-dessus est identique à celle du menu tableau de bord (Administration) sauf que dans ce tableau la colonne des pièces justificatives n'est plus disponible. Ce tableau permet d'avoir un vue globale des opérations effectués sur les matériels et équiments au sein d'une direction. Les détails sur l'opération sont disponible en cliquant sur l'icone contenu dans la colonne Détail.

4.9 - Liste des matériels, de leurs états et de leurs historiques

Menu accéssible pour les personnels du Dépositaire Comptable (DC), Gestionnaires d'Activités (GAC), Consultation, Direction, Administrateur, Administrateur général . Ce menu présente la liste de tous les matériels de la direction avec leur états ainsi que l' historique des mouvements effectués.

Ce tableau présente les informations suivantes :

    ID : Identifiant du matériel

    Numéro série : Numéro de série du matériel

    Type de matériel : Type du matériel

    Codification : Code unique du matériel

    Nomenclature : Nomeclature associée au matériel

    Etat : Etat du matériel

    Marque : Marque du matériel

    Dépositaire : Dépositaire du matériel

    Détail : Colonne contenant une icone permettant d'afficher les détails concernant le matériel

    Statut : Colonne contenant une icone permettant d'afficher le statut du matériel

    Historique : Colonne contenant une icone permettant d'afficher l'historique des mouvements effectués avec le matériel

    Statut : Colonne à valeur binaire (Validé ou Non validé)

    Détenteur : Colonne à valeur binaire (Oui ou Non)
